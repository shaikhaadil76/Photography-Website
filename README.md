# 📸 Photography Website

This is A simple photography website built using HTML and CSS to practice responsive layouts, sidebar navigation, and modern web design.

A simple Photography Website built using **HTML** and **CSS**.

## 📖 About the Project

This project was created to practice frontend development and improve my HTML and CSS skills. It features a clean design with a sidebar navigation and a photography gallery.

## ✨ Features

- Clean and simple UI
- Sidebar navigation
- Photography gallery
- Modern webpage layout
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎯 What I Learned

- HTML page structure
- CSS styling
- Flexbox
- Website layout design
- Working with images

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/shaikhaadil76/Photography-Website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
